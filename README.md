# Project-1
The project about fire alarms 
 Fire Alarm System Project
 1. Introduction
A fire alarm system is a safety device used to detect fire hazards such as smoke, heat, or gas and alert people through alarms. It helps prevent accidents and saves lives by giving early warning.
 2. Problem Statement
Fire accidents in homes, industries, and public places can cause severe damage to life and property. Many times, fires are detected too late due to lack of proper monitoring systems.
 3. Proposed Solution
To design a smart fire alarm system using sensors and microcontrollers that can detect smoke, temperature rise, or gas leakage and alert users immediately using a buzzer or mobile notification.
 4. Components Used
Arduino Uno / NodeMCU – Main controller
MQ-2 Smoke Sensor – Detects smoke and gas
DHT11 Sensor – Measures temperature and humidity
Buzzer – Produces alarm sound
LED – Visual alert
Relay Module – Controls external devices (fan, water pump)
Power Supply
(Optional) Blynk App – Mobile notification
 5. Working Principle
The MQ-2 sensor detects smoke or gas in the environment.
The DHT11 sensor monitors temperature changes.
If smoke level or temperature exceeds a threshold:
Buzzer turns ON
LED glows
Notification sent to mobile (if IoT-based)
Relay can activate safety measures like water sprinklers.
 6. Key Features
Early fire detection
Real-time alerts
Low cost and easy to install
IoT-based remote monitoring (optional)
Automatic response system
 7. Product Overview
This system is a compact and efficient fire safety solution designed for homes, offices, and industries. It continuously monitors environmental conditions and provides instant alerts in case of danger.
 8. Advantages / Benefits
Prevents major fire accidents
Saves lives and property
Low maintenance
Easy to operate
Can be integrated with smart home systems
 9. Applications
Homes and apartments
Factories and industries
Schools and colleges
Hospitals
Warehouses
 10. Limitations
Sensors may give false alarms in some conditions
Requires continuous power supply
Limited detection range
 11. Future Enhancements
AI-based fire prediction
Integration with emergency services
GSM module for SMS alerts
Automatic fire extinguishing system
12. Conclusion
The fire alarm system is an essential safety solution that helps in early detection and prevention of fire accidents. By using sensors and microcontrollers, this project provides a reliable, affordable, and efficient way to enhance safety in various environments.
